LazyHunter Pro v2.0
Elite Web Application & Infrastructure Recon Assistant

LazyHunter Pro is a high-speed, asynchronous reconnaissance tool designed for bug hunters to quickly map the attack surface of a target IP or range. It leverages Shodan's InternetDB for lightning-fast port and vulnerability discovery and the CVEDB API for deep contextual analysis.

Asynchronous Processing: Scans hundreds of targets simultaneously using asyncio and aiohttp.


Intelligent CVE Caching: Minimizes API calls by caching CVE details locally, preventing rate-limiting.


Exploit Correlation: Generates direct links to Exploit-DB for every identified vulnerability to speed up Proof-of-Concept (PoC) development.

Nuclei Ready: Automatically generates a nuclei_targets.txt file (formatted as IP:Port) for immediate active scanning.

Hostname Resolution: Identifies reverse DNS records to help prioritize high-value assets (e.g., prod vs dev servers).
